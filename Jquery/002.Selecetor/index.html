Jquery Selector là gì?

CSS selector là bộ chọn CSS các thẻ html mà bạn mong muốn, ví dụ: .class, #id , tag_name ( body , div , p ... )

Jquery selector là bộ chọn của Jquery để chọn các thẻ html mà bạn mong muốn

        CSS                   =>         Jquery

        .class                =>        $(".class")
        #id                   =>        $("#id")
        tag_name(div)         =>     $("tag_name" $("div")
        .class A .class B     =>     $(".class A .class B")
        .class A > class B    =>     $(".class A > .class B")

        CHÚ Ý: tất cả các selector tên class, yên id, tê thẻ đều phải nằm
        trong "" hoặc '' nhưng riêng document và window thì k cần.
